## git 계정 설정.

```bash
$ git config --global user.name "name"
$ git config --global user.email name@example.com
```

- '--global' 은 전역 설정
- '--local' 은 프로젝트내 설정

## git 설정 내용 보기.

```bash
$ git config --list
```

참고 : [git book](https://git-scm.com/book/ko/v2)
