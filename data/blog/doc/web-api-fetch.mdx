## fetch

fetch 함수는 HTTP 요청 전송 기능을 제공하는 클라이언트 사이드 Web API다.
비교적 최근에 추가된 Web API라서 인터넷 익스플로러를 제외한 대부분의 브라우저에서 제공한다.

- [can i use - fetch](https://caniuse.com/?search=fetch)

```javascript
// const promise = fetch(url, [, options]);

fetch('https://jsonplaceholder.typicode.com/todos/1')
  .then((response) => console.log(response))
  .catch((err) => console.log(err));
```

```javascript
fetch('https://jsonplaceholder.typicode.com/todos/1')
  .then((response) => response.json())
  .then((json) => console.log(json));
```

fetch 함수는 HTTP 응답을 나타내는 Response 객체를 래핑한 프로미스를 반환하므로 후속 처리 메서드 then을 통해 Response객체를 전달받을 수 있다.

- 예시

```typescript
interface Todo {
  userId: number;
  id: number;
  title: string;
  completed: boolean;
}

// 기본적인 GET 요청
async function fetchTodo(): Promise<void> {
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/todos/1');

    // 응답이 성공적으로 받아진 경우
    if (!response.ok) {
      throw new Error('Network response was not ok');
    }

    const data: Todo = await response.json();
    console.log('Todo Title:', data.title);
  } catch (error) {
    // 오류 처리
    console.error('Error:', error);
  }
}

fetchTodo();
```

```javascript
// fetch api 커스텀훅 예시
const useFetch = () => {
  const [response, setResponse] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  const fetchData = async (url, options) => {
    try {
      const result = await fetch(url, options);
      const json = await result.json();

      if (!result.ok) {
        throw new Error(json.message || 'Fetch failed');
      }

      setResponse(json);
    } catch (error) {
      setError(error);
    } finally {
      setLoading(false);
    }
  };

  const get = (url) => fetchData(url, { method: 'GET' });
  const post = (url, data) => fetchData(url, { method: 'POST', body: JSON.stringify(data) });
  const put = (url, data) => fetchData(url, { method: 'PUT', body: JSON.stringify(data) });
  const patch = (url, data) => fetchData(url, { method: 'PATCH', body: JSON.stringify(data) });
  const remove = (url) => fetchData(url, { method: 'DELETE' });

  return { response, loading, error, get, post, put, patch, remove };
};

export default useFetch;
```

```javascript
// useFetch 훅 사용 예시
import React, { useEffect } from 'react';
import useFetch from './useFetch';

const ExampleComponent = () => {
  const apiUrl = 'https://jsonplaceholder.typicode.com/todos/1';
  const { response, loading, error, get, post, put, patch, remove } = useFetch();

  useEffect(() => {
    // GET 요청
    get(apiUrl);

    // POST 요청
    const postData = { userId: 1, title: 'Buy groceries', completed: false };
    post(apiUrl, postData);

    // PUT 요청
    const putData = { userId: 1, title: 'Buy groceries', completed: true };
    put(apiUrl, putData);

    // PATCH 요청
    const patchData = { completed: true };
    patch(apiUrl, patchData);

    // DELETE 요청
    remove(apiUrl);
  }, [apiUrl, get, post, put, patch, remove]);

  if (loading) {
    return <p>Loading...</p>;
  }

  if (error) {
    return <p>Error: {error.message}</p>;
  }

  return (
    <div>
      <h1>Data from Placeholder API</h1>
      <pre>{JSON.stringify(response, null, 2)}</pre>
    </div>
  );
};

export default ExampleComponent;
```
